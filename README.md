# JobPilot

JobPilot is an AI-powered job hunting assistant built with Next.js. It helps users create a profile, upload a resume, discover relevant tech jobs, and research companies before applying.

## Features

- AI-assisted job search and matching
- Resume upload and profile management
- Job details and company research flow
- Dashboard with analytics and recent activity
- OAuth sign-in and profile-based recommendations

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- PostHog analytics
- InsForge / Browserbase / Stagehand integration

## Getting Started

1. Install dependencies:
   npm install
2. Start the development server:
   npm run dev
3. Open http://localhost:3000

## Available Scripts

- npm run dev — start the app locally
- npm run build — create a production build
- npm run start — run the production build
- npm run lint — run ESLint
