Contributing

Thank you for contributing! Please follow these guidelines:

- Run `npm run lint` and `npm run format` before opening a PR.
- Keep changes small and focused; follow the repository's coding standards in `context/code-standards.md`.
- Add tests for new behavior and update docs in `docs/`.

Branching

- Create feature branches from `main` named `feature/short-description`.
- Open pull requests targeting `main` with a clear title and description.

Local setup

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
